ТЗ Разработать html-страницу для сайта для фильмов.

Сайт должен состоять из такой структуры
- Блок с навигацией по сайту
- Основной блок с контентом
- Подвал сайта (в нем только копирайт (через мнемонику) на вас)

Навигация состоит из 5 ссылок
- главная
- топ-50
- топ по рейтингу Валеры
- новости
- контакты (при клике на контакты - перекидывает на эту секцию)

Основной блок с контентом будет размещать в себе 2 логических блока
- секция с карточками фильмов (карточек 3 штуки)
- секция контактов

карточка фильма состоит из
- заголовок
- его рейтинг (к примеру 5.3/10)
- дата выхода в прокат (в формате день, месяц и год)
- картинка фильма
- краткое описание фильма (любой текст, который хотите)
- список актеров (не более трех) с ссылками на них на википедию
- кнопка поставить лайк (картинка лайка и слово "лайк")
- теги на этот фильм (к примеру #valera, #dramma, #ololo, etc)

секция о нас состоит из:
- ваши контакты (телефон, почта)
